What is Python Turtle?
Your digital paintbrush

Imagine you have a tiny turtle 🐢 on your screen holding a pen. You can tell it to move forward, turn left, change colour, and draw all sorts of shapes. That's exactly what Python's Turtle library does!

🌍 Real Life Example
Think of the turtle like an Etch-A-Sketch toy or a robot with a marker attached to its belly. Wherever the robot walks, it draws a line. You control the robot with code!

🖥️ The Screen

Like a blank piece of paper or a whiteboard. We set its size, title, and background colour.

🐢 The Turtle (Pen)

A small arrow/turtle shape. We move it around and it draws on the screen.

📍 Understanding Coordinates
The centre of the screen is (0, 0). Going right increases X. Going up increases Y. Think of it like a map: East = +X, North = +Y.

📁 Project 1.1 – Color & Text

Setting up a screen with a pink background and writing "Welcome" in navy blue using a fancy font.

Python 
''' Canvas has default size (700x650), white background, pensize 1 '''
import turtle

# Create a screen and set it up
scr = turtle.Screen()
scr.setup(800, 600)         # width=800, height=600
scr.title("Playing with colors")
scr.bgcolor('light pink')     # background colour

# Create a turtle (the pen)
t = turtle.Turtle()
t.penup()                     # don't draw while moving
t.goto(-100, 200)            # move to position
t.color('#000080')           # hex colour: navy blue
t.write("Welcome ", font=('bradley hand itc', 40, "bold"))

# Another turtle writing at centre
t1 = turtle.Turtle()
t1.pu()
t1.goto(0, 0)
t1.pd()
t1.write("Welcome", move=True, align="center",
         font=("Arial", 8, "normal"))

turtle.done()

🔢 What is RGB?
RGB stands for Red, Green, Blue. Every colour on a screen is a mix of these three. Like mixing paints — but instead of physical paint, we mix light! RGB(255,0,0) = pure red. RGB(0,255,0) = pure green. RGB(0,0,255) = pure blue. RGB(255,255,255) = white. RGB(0,0,0) = black.

Polygons
Drawing any polygon using the formula: 360 ÷ sides

A polygon is any shape with straight sides. The magic formula is:
Turn Angle = 360 ÷ number of sides

🌍 Real Life Example
Imagine you're marching in a circle. If you need to complete the circle in 6 equal turns (hexagon), each turn is 360 ÷ 6 = 60°. If you do it in 3 turns (triangle), each is 120°. Same idea works for any polygon!

Random Stars
Nested loops, setheading, and random placement

Stars are drawn using a neat trick: move forward and turn by 144° five times! We'll draw rows and columns of stars, then scatter them randomly using the random module.

🌍 Real Life Example
Think of printing stickers on a sheet. First row (horizontal), then column by column (vertical), then you scatter them randomly like confetti at a party! Each "sticker" = one star.
⭐ How to Draw One Star

A 5-pointed star needs 5 forward moves with a 144° right turn after each.

Star Logic 
for i in range(5):
    t.forward(50)
    t.right(144)   # 144° gives a 5-pointed star

📁 Project 5.3 – Twinkling Stars (Random)

Scatter 50 colourful filled stars at random positions using randint for position and choice for colour!

Python 
import turtle
from random import randint, choice

colors = ["red", "blue", "orange", "pink",
          "indigo", "violet", "green", "yellow"]

s = turtle.Screen()
s.title('Twinkling Stars')
s.setup(800, 600)
s.bgcolor("lightgrey")
t = turtle.Turtle()

for i in range(50):            # 50 random stars
    t.color(choice(colors))   # random colour from list
    x = randint(-400, 400)    # random x position
    y = randint(-300, 300)    # random y position
    turns = 6
    t.begin_fill()
    while turns > 0:
        t.forward(25)
        t.left(145)
        turns -= 1
    t.end_fill()
    t.penup()
    t.goto(x, y)
    t.pendown()

s.exitonclick()

Functions
Reusable blocks of code with def

A function is a named block of code that you can run (call) whenever you need it. Instead of writing the same code 5 times, you write it once inside a function and call it 5 times!

🌍 Real Life Example
Think of a function like a recipe card 📋. The recipe for "make tea" is written once. Whenever you want tea, you just follow that one recipe card instead of writing out all the steps again. In Python, def make_tea(): is your recipe card!
Function Syntax 
def function_name(parameter1, parameter2):
    # code goes here
    # this is the function BODY

# Calling the function:
function_name(value1, value2)

📁 Project 6.2 – Smiley Face with Functions 😊

Notice in Project 3.2 (Panda), we moved the turtle to many positions to draw dots. We kept repeating pu, goto, pd, dot. Let's create a function c_shape() to avoid repetition!

Python 
import turtle
scr = turtle.Screen()
scr.setup(800, 500)
scr.title("Smiley")
t = turtle.Turtle()

# Reusable function to draw a dot at any position!
def c_shape(x, y, size, color):
    t.pu()
    t.goto(x, y)
    t.pd()
    t.dot(size, color)

# Ears
c_shape(-180, 180, 140, 'black')
c_shape(180, 180, 140, 'black')

# White face
t.pu(); t.goto(0, -200); t.pd()
t.fillcolor("white")
t.begin_fill()
t.circle(200)
t.end_fill()

# Eyes (black outer + white inner)
c_shape(-100, 20, 70, 'black')
c_shape(100, 20, 70, 'black')
c_shape(-100, 20, 30, 'white')
c_shape(100, 20, 30, 'white')

# Nose
c_shape(0, -80, 50, 'black')

# Smile (two semicurves)
t.goto(0, -100)
t.rt(90)
for i in range(180):
    t.fd(1); t.lt(1)

t.pu(); t.goto(0, -100); t.rt(180); t.pd()
for i in range(180):
    t.fd(1); t.rt(1)

t.ht()
turtle.done()
✅ Why Functions are Awesome
Without the c_shape() function, we'd write pu → goto → pd → dot seven times = 28 lines! With the function, calling it 7 times = 7 lines. Less code, less mistakes, easier to read!

Fidget Spinner
Keyboard events, global variables, animation timer

This is our first interactive animation! Press SPACE to spin. The spinner gradually slows down like a real fidget spinner. We learn key binding, global variables, and timers.

🌍 Real Life Example
Like a real fidget spinner toy: you flick it (press space = flick), it spins fast, then slowly stops. In code, we add to speed when you press space and subtract a little every frame until it reaches zero!

New Concepts

⌨️ Key Binding

scr.listen() tells Python to watch for key presses.
scr.onkeypress(func, "space") calls func when space is pressed.

🌐 Global Variables

A variable created outside any function. To change it inside a function, you must write global variable_name first.

⏱️ Timer

scr.ontimer(func, ms) calls func after ms milliseconds. This creates smooth animation!

📁 Project 7 – Fidget Spinner

Python 
import turtle

scr = turtle.Screen()
scr.bgcolor("pink")
scr.title("Fidget Spinner")
scr.setup(800, 600)

t = turtle.Turtle()
t.pensize(20)
t.color("yellow")

def spinner():
    t.clear()             # erase previous frame
    angle = turn / 10
    t.right(angle)
    # Draw 3 blades
    t.forward(100); t.dot(120, "red");   t.backward(100); t.right(120)
    t.forward(100); t.dot(120, "blue");  t.backward(100); t.right(120)
    t.forward(100); t.dot(120, "green"); t.backward(100); t.right(120)
    scr.update()

def flick():
    global turn
    turn += 10          # pressing space adds spin speed

def animate():
    global turn
    if turn > 0:
        turn -= 1       # slowly reduce speed (friction)
    spinner()
    scr.ontimer(animate, 20)  # call animate every 20ms

turn = 0                    # global variable
scr.tracer(False)           # hide drawing steps for speed
scr.listen()
scr.onkeypress(flick, "space")
animate()
turtle.done()

Snake Game
Direction movement, collision, growing body

Our first real game! The snake moves in a direction you control with arrow keys. It eats food and grows longer. If it hits the wall or itself, it resets!

🌍 Real Life Example
Like the classic Nokia phone game! The snake (a line of squares) moves around. It eats dots (food) and its tail gets longer. If it crashes into a wall or its own body, game over!

Key Concepts in Snake Game

📍 xcor() and ycor()

Get the turtle's current x or y position.
t.xcor() → x coordinate
t.ycor() → y coordinate

📏 distance()

t.distance(f) returns how many pixels apart two turtles are. We use this to detect when snake eats food (< 20 pixels).

📋 List for Body

segments = [] stores all body pieces. Each time food is eaten, a new turtle square is added!

🔄 Body Movement

Each segment moves to where the segment in front of it was. Segment N → goes to position of Segment N-1. Like a line of people copying the person ahead!

📁 Project 8 – Snake Game

Python 
import turtle, random, time

scr = turtle.Screen()
scr.setup(600, 600)
scr.bgcolor("pink")
scr.title("Snake game")
scr.tracer(False)

# Snake head
t = turtle.Turtle()
t.penup(); t.shape("square"); t.color("white")
t.direction = "stop"    # custom attribute on turtle object

# Food
f = turtle.Turtle()
f.shape("circle"); f.color("green"); f.penup()
f.goto(random.randint(-250, 250), random.randint(-250, 250))

# Direction functions
def up():    t.direction = "up"
def down():  t.direction = "down"
def left():  t.direction = "left"
def right(): t.direction = "right"

def move():
    if t.direction == "up":    t.sety(t.ycor() + 20)
    if t.direction == "down":  t.sety(t.ycor() - 20)
    if t.direction == "right": t.setx(t.xcor() + 20)
    if t.direction == "left":  t.setx(t.xcor() - 20)

scr.listen()
scr.onkey(up, "Up"); scr.onkey(down, "Down")
scr.onkey(left, "Left"); scr.onkey(right, "Right")

segments = []

while True:
    scr.update()
    # Eat food
    if t.distance(f) < 20:
        f.goto(random.randint(-250, 250), random.randint(-250, 250))
        new_seg = turtle.Turtle()
        new_seg.penup(); new_seg.shape("square"); new_seg.color("orange")
        segments.append(new_seg)

    # Move body segments (from tail to head)
    for i in range(len(segments) - 1, 0, -1):
        x = segments[i - 1].xcor()
        y = segments[i - 1].ycor()
        segments[i].goto(x, y)
    if len(segments) > 0:
        segments[0].goto(t.xcor(), t.ycor())

    # Wall collision — reset game
    if t.xcor() > 290 or t.xcor() < -290 or t.ycor() > 290 or t.ycor() < -290:
        t.goto(0, 0); t.direction = "stop"
        for seg in segments: seg.goto(1000, 1000)
        segments.clear()

    # Self collision
    for seg in segments:
        if seg.distance(t) < 20:
            t.goto(0, 0); t.direction = "stop"

Angle Calculations (The Math!)

HandFormulaWhy?
Second handsec × 6360° ÷ 60 seconds = 6° per second
Minute handmin × 6 + sec × 0.16° per minute + smooth movement
Hour handhr × 30 + min × 0.530° per hour + smooth movement
Python 
import turtle, datetime, time, math

screen = turtle.Screen()
screen.bgcolor("black"); screen.title("Analog Clock"); screen.setup(600, 600)

clock = turtle.Turtle()
clock.speed(0); clock.pensize(5); clock.color("white"); clock.hideturtle()

def draw_clock_face():
    clock.penup(); clock.goto(0, -250); clock.pendown()
    clock.circle(250)
    for hour in range(12):
        clock.penup(); clock.goto(0, 0)
        clock.setheading(-30 * hour + 60)
        clock.forward(220); clock.pendown(); clock.forward(20); clock.penup()

min_hand  = turtle.Turtle(); min_hand.color("blue");  min_hand.pensize(3);  min_hand.hideturtle()

    t.setheading(90); t.right(angle)
    t.pendown(); t.forward(length)

def update():
    now = datetime.datetime.now()
    sec = now.second; minute = now.minute; hour = now.hour % 12
    sec_angle  = sec * 6
    min_angle  = minute * 6 + sec * 0.1

    draw_hand(min_hand, 150, min_angle)

        if ch == word[i]:      color = "green"   # right spot
        elif ch in word:       color = "yellow"  # wrong spot
        else:                  color = "gray"    # not in word
        draw_square(x, row_y, color); write_letter(x, row_y, ch)
