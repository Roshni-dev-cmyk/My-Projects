⚔️ Project Overview

Two teams fight 6 rounds. Each round randomly picks one hero from each team and randomly decides the winner. At the end, the team that won the most rounds is declared champion.

7.Marvel vs DC/marvelVSdc.py 
import random

avengers      = ['Iron Man', 'Captain America', 'Black Widow',
                  'Hawkeye', 'Hulk', 'Thor']
justice_league = ['Aquaman', 'Batman', 'Flash',
                   'Green Lantern', 'Superman', 'Wonder Woman']
teams = ['avengers', 'justice_league']

print('------- AVENGERS vs JUSTICE LEAGUE -------')
win = []
for round in range(6):
    print(f"Round {round+1}: {random.choice(avengers)} vs {random.choice(justice_league)}")
    winner = random.choice(teams)
    print(f"Winner: {winner}")
    win.append(winner)

# Overall winner:
if win.count('avengers') > win.count('justice_league'):
    print("🏆 Avengers win!")
else:
    print("🏆 Justice League wins!")

🪢 Project — Hangman

🎮 How it Works

  • Computer picks a random word from a list
  • Display dashes (one per letter) instead of the word
  • Player guesses letters one at a time
  • Correct guess → reveal the letter in position
  • Wrong guess → show hangman body part, reduce chances
  • Player wins if they guess all letters before 7 wrong guesses
8.Hangman/hangman_final.py 
import random
# hangman ASCII art stages (indices 0–6)
hangman1 = [r"""
 +---+
     |
     |
     
    ==="", ...  # 7 stages total]

words = ["apple", "orange", "grapes", "banana", "kiwi"]
word   = random.choice(words)
guessedletters = ""
chances = 7

print("Guess the Fruit | You have", chances, "chances")
for i in word:
    print("-", end=" ")

while chances > 0:
    guess = input("Enter a letter: ")
    guessedletters += guess
    wrong = 0
    for i in word:
        if i in guessedletters:
            print(i, end=" ")
        else:
            print("-", end=" ")
            wrong = 1
    print()
    if wrong == 0:
        print("Correct! You won!"); break
    if guess not in word:
        print("Wrong guess")
        print(hangman1[7-chances])
        chances -= 1

Cryptography Caesar cipher · functions · encrypt & decrypt

🔐 What is Cryptography?

Cryptography (from Greek: "hidden writing") is the art of converting information into an unreadable format so only the intended recipient can read it.

Encrypt = convert readable text → unreadable code

Decrypt = convert code → readable text back

🌍 Real Life Every time you log in to a website, your password is encrypted. WhatsApp messages use end-to-end encryption.

🔢 Functions

A function is a named block of code that performs one specific task. Define it once, call it many times!

9.Cryptography/cryptography2.py — Encrypt & Decrypt 
letters = ['a','b','c','d',...,'z']  # full 26-letter alphabet list

def encrypt(text, n):
    result = []
    for x in text:
        if x == " ":
            result.append(" ")
        else:
            converted = (letters.index(x) + n) % 26
            result.append(letters[converted])
    print(''.join(result))   # join() converts list → string

def decrypt(text, n):
    result = []
    for i in text:
        if i == " ":
            result.append(" ")
        else:
            converted = (letters.index(i) - n) % 26
            result.append(letters[converted])
    print(''.join(result))

playing = ''
while playing != 'quit':
    text   = input('Enter the text: ').lower()
    key    = int(input("Enter the key: "))
    choice = input('Encrypt or decrypt? ').lower()
    if   choice == 'encrypt': encrypt(text, key)
    elif choice == 'decrypt': decrypt(text, key)
    playing = input('Enter quit to stop: ')
💡 How it worksEach letter shifts forward by the key number. "a" with key 3 becomes "d". We use % 26 to wrap around from "z" back to "a".

🔁 Recursion — Factorial

🌀 What is Recursion?

A function that calls itself to solve a smaller version of the same problem. Every recursive function needs a base case (when to stop) and a recursive case (calls itself).

🌍 Real Life Russian nesting dolls. A folder inside a folder inside a folder. A mirror reflecting another mirror.
12.extras/Recursion/factorial.py 
def fact(num):
    if num == 1:              # BASE CASE
        return num
    else:
        return num * fact(num - 1)  # RECURSIVE CASE

num = int(input("Enter a number: "))
if num < 0:
    print("Factorial of negative numbers cannot be found")
elif num == 0:
    print("Factorial of 0 is 1")
else:
    print("Factorial of", num, "is", fact(num))
💡 Example: fact(4)
fact(4) = 4 × fact(3) = 4 × 3 × fact(2) = 4 × 3 × 2 × fact(1) = 4 × 3 × 2 × 1 = 24

12.extras/wordle.py 
import random
words = ["alive", "bench", "begin", "alarm"]
word  = random.choice(words)
for i in range(6):            # 6 guesses allowed
    guess = input("Enter a 5-letter word: ")
    if guess == word:
        print(f"\033[42m{word}\033[0m")   # green highlight
        break
    elif len(guess) < 5: print("Too short")
    elif len(guess) > 5: print("Too long")
    else:
        for j in guess:
            if j in word:
                if word.index(j) == guess.index(j):
                    print(f"\033[42m{j}\033[0m", end=" ")  # green (right place)
                else:
                    print(f"\033[43m{j}\033[0m", end=" ")  # yellow (wrong place)
            else:
                print(j, end=" ")                    # grey (not in word)
    print()
💡 Color Codes\033[42m = green background, \033[43m = yellow background, \033[0m = reset color. Used in terminal output!
